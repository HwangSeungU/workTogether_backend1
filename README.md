## ✨ 주요 기능 목록

### 1. 정보 제공 및 탐색
* **가게 정보:** 가게 리스트 및 세부 페이지, 찜 하기
* **위치 정보:** 지도 페이지, 지하철 페이지 (편의 시설 정보 포함)
* **뉴스:** 일반 뉴스 및 배너 뉴스 제공

### 2. 커뮤니티 및 소통
* **일반 게시판:** 게시글 작성/조회, 댓글 기능, 게시글 신고 기능
* **돌봄 회원:** 리스트 및 세부 페이지 (돌봄 서비스 관련)
* **쪽지:** 회원 간 쪽지 보내기 기능

### 3. 사용자 및 시스템 관리
* **회원 기능:** 회원가입, 로그인, 아이디 찾기, 일반 회원/돌봄 회원 마이페이지
* **관리자 페이지:**
    * 회원 관리, 공지사항 관리, 1대1 문의 관리
    * 게시글 신고 관리
    * 돌봄 회원 환급/신청 관리
    * 뉴스/배너 뉴스 관리
